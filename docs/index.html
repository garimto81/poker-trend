핵심 기능

Gemini AI 심층 분석

Google Gemini AI가 수집된 데이터를 분석하여 트렌드 패턴을 파악하고 미래를 예측합니다.

  • 현재 트렌드 패턴 인식
  • 1주-1개월 미래 예측
  • 실행 가능한 전략 제안
  • 한국어 자동 번역

체계적 데이터 수집

YouTube Data API v3를 활용하여 8개 핵심 키워드의 상위 콘텐츠를 실시간 추적합니다.

  • 키워드당 TOP 5 영상
  • 조회수, 좋아요, 댓글 분석
  • 채널별 통계 집계
  • 시간대별 패턴 분석

Slack 자동 리포팅

분석 결과를 깔끔하게 정리하여 지정된 시간에 Slack 채널로 자동 전송합니다.

  • 하이퍼링크 포함 리포트
  • 이모지로 가독성 향상
  • AI 인사이트 포함
  • 실패 시 자동 알림

GitHub Actions 자동화

서버 없이 GitHub Actions로 완전 자동화된 워크플로우를 운영합니다.

  • 서버리스 아키텍처
  • 무료 티어 활용
  • 자동 스케줄링
  • 수동 실행 지원

스마트 번역 시스템

영어 콘텐츠를 자연스러운 한국어로 번역하여 국내 사용자에게 최적화된 정보를 제공합니다.

  • 단일 번역 보장
  • 문맥 기반 번역
  • 실패 시 원본 보존
  • 특수문자 자동 정리

3단계 리포팅

일간, 주간, 월간 각각 다른 관점으로 트렌드를 분석하여 종합적인 인사이트를 제공합니다.

  • 일간: 즉각적 트렌드
  • 주간: 성장률 분석
  • 월간: 장기 예측
  • 맞춤형 분석 깊이

자동 실행 스케줄

평일 10:00
일간 리포트
화요일~금요일 오전 10시에 최근 24시간 트렌드 분석
월요일 11:00
주간 리포트
매주 월요일 오전 11시에 지난 7일간 트렌드 종합 분석
첫째주 월요일 14:00
월간 리포트
매월 첫째주 월요일 오후 2시에 지난 30일 트렌드 심층 분석
