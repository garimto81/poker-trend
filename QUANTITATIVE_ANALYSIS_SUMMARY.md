## 📊 핵심 정량적 지표 개발

### 1. 참여율 (Engagement Rate)
```
참여율 = (좋아요 + 댓글) / 조회수
```

### 2. 바이럴 점수 (Viral Score)  
```
바이럴점수 = log₁₀(조회수) × 0.4 + 
           참여율×1000 × 0.3 + 
           log₁₀(좋아요) × 0.2 + 
           log₁₀(댓글) × 0.1
```

### 3. 트렌드 모멘텀 (Trend Momentum)
```
트렌드모멘텀 = 키워드상대성과 × 개별비디오성과
```

### 4. 성과 백분위 (Performance Percentile)
```
성과백분위 = (조회수백분위 + 참여율백분위) / 2
```
