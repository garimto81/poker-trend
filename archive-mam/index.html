GFX 오버레이 학습기

STEP 1

포커 방송의 RFID GFX 오버레이 패턴을 학습시킵니다.

  • 영상을 재생하며 GFX/Game 구분 마킹
  • 색상 균일도, 텍스트 밀도 등 특징 자동 추출
  • 최소 10개 샘플로 높은 정확도 달성
  • 학습된 모델을 로컬에 저장
🎰 GFX 학습 시작
🃏

포커 핸드 분류기

STEP 2

학습된 GFX 모델로 포커 핸드를 자동 분류합니다.

  • GFX 출현 - 15초 = 핸드 시작점
  • GFX 소멸 + 15초 = 핸드 종료점
  • 실시간 분석 진행률 표시
  • JSON 형식으로 결과 내보내기
🃏 핸드 분류 시작

⏱️ 15초 규칙이란?

포커 방송에서 RFID GFX 오버레이는 핸드가 시작된 후 약 15초 뒤에 나타나고, 핸드가 끝나기 약 15초 전에 사라집니다.

  • 핸드 시작 = GFX 출현 시점 - 15초
  • 핸드 종료 = GFX 소멸 시점 + 15초

이 규칙을 활용하면 GFX 감지만으로도 정확한 핸드 경계를 결정할 수 있습니다.

⏱️

🚀 빠른 시작 가이드

Step 1: GFX 패턴 학습
  1. GFX 오버레이 학습기를 엽니다
  2. 포커 영상을 업로드합니다
  3. 영상을 재생하며 GFX가 보이면 'GFX' 버튼 클릭
  4. 실제 게임 화면이면 'Game' 버튼 클릭
  5. 10개 이상 샘플 수집 후 '모델 학습' → '모델 저장'
Step 2: 핸드 자동 분류
  1. 포커 핸드 분류기를 엽니다
  2. 'GFX 모델 로드' 버튼으로 학습된 모델 불러오기
  3. 분석할 포커 영상 업로드
  4. '분석 시작' 클릭하고 완료 대기
  5. 감지된 핸드 목록을 JSON으로 다운로드
