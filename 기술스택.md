## 3. GitHub Actions 워크플로우

### 3.1 데이터 수집 워크플로우
```yaml
name: Collect Trend Data
on:
  schedule:
    - cron: '0 * * * *'  # 매시간
  workflow_dispatch:

jobs:
  collect:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-python@v5
      - run: pip install -r requirements.txt
      - run: python scripts/collect_trends.py
      - uses: actions/upload-artifact@v4
```

### 3.2 콘텐츠 생성 워크플로우
```yaml
name: Generate Content
on:
  repository_dispatch:
    types: [trend-detected]

jobs:
  generate:
    runs-on: ubuntu-latest
    steps:
      - name: Generate Script
        run: python scripts/generate_script.py
      - name: Create Video
        run: python scripts/create_video.py
      - name: Upload to Release
        uses: softprops/action-gh-release@v2
```

### 3.3 배포 워크플로우
```yaml
name: Deploy to Platforms
on:
  release:
    types: [published]

jobs:
  deploy:
    runs-on: ubuntu-latest
    strategy:
      matrix:
        platform: [youtube, tiktok, instagram]
    steps:
      - name: Upload to ${{ matrix.platform }}
        run: python scripts/upload_${{ matrix.platform }}.py
```

  - uploaders/: 플랫폼 업로드 스크립트
