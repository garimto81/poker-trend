# 포커 트렌드 쇼츠 제작 기술 스택 설계

## 1. 전체 시스템 아키텍처

```
┌─────────────────────────────────────────────────────────────────┐
│                        Frontend Dashboard                       │
│                     (React + TypeScript)                       │
└─────────────────────────────────────────────────────────────────┘
                                   │
┌─────────────────────────────────────────────────────────────────┐
│                        API Gateway                             │
│                      (Node.js + Express)                       │
└─────────────────────────────────────────────────────────────────┘
                                   │
        ┌──────────────────────────┼──────────────────────────┐
        │                          │                          │
┌───────▼────────┐     ┌──────────▼──────────┐     ┌──────────▼──────────┐
│ Data Collection │     │ Content Generation  │     │ Distribution Service│
│    Service      │     │      Service        │     │                     │
│  (Python/Node)  │     │   (Python + AI)     │     │   (Node.js + APIs)  │
└────────────────┘     └─────────────────────┘     └─────────────────────┘
        │                          │                          │
┌───────▼────────┐     ┌──────────▼──────────┐     ┌──────────▼──────────┐

└────────────────┘     └─────────────────────┘     └─────────────────────┘
```

## 2. 핵심 기술 스택

### 2.1 백엔드 서비스
```yaml
언어:
  - Python 3.11+: AI/ML 처리, 데이터 분석
  - Node.js 18+: API 서버, 실시간 처리
  - TypeScript: 타입 안정성

프레임워크:
  - FastAPI: Python API 서버
  - Express.js: Node.js 웹 서버
  - Socket.io: 실시간 통신

AI/ML 라이브러리:
  - OpenAI GPT-4: 스크립트 생성
  - Transformers: 감정 분석, 텍스트 처리
  - spaCy: 자연어 처리
  - scikit-learn: 트렌드 분석
```

### 2.2 데이터베이스
```yaml
PostgreSQL:
  - 트렌드 데이터 저장
  - 사용자 관리
  - 성과 메트릭

MongoDB:
  - 콘텐츠 템플릿
  - 영상 메타데이터
  - 로그 데이터

Redis:
  - 캐싱
  - 세션 관리
  - 작업 큐
```

### 2.3 프론트엔드
```yaml
React 18+:
  - 컴포넌트 기반 UI
  - TypeScript 적용

상태관리:
  - Zustand: 전역 상태
  - React Query: 서버 상태

UI 라이브러리:
  - Ant Design: 관리자 대시보드
  - Chart.js: 데이터 시각화
  - Framer Motion: 애니메이션
```

### 2.4 영상 처리
```yaml
FFmpeg:
  - 영상 편집 자동화
  - 포맷 변환
  - 썸네일 생성

ImageMagick:
  - 이미지 처리
  - 텍스트 오버레이

TTS (Text-to-Speech):
  - Azure Cognitive Services
  - Google Cloud TTS
  - ElevenLabs API
```
